## 模型选择建议

- 如果不确定，使用 `gpt-5.4`（旗舰模型，适合复杂推理和编码）。
  If unsure, use `gpt-5.4` (flagship model, suitable for complex reasoning and coding).
- 如果优化延迟和成本，选择 `gpt-5.4-mini` 或 `gpt-5.4-nano`。
  If optimizing for latency and cost, choose `gpt-5.4-mini` or `gpt-5.4-nano`.
- 所有最新模型支持文本、图像输入、文本输出、多语言和视觉。
  All latest models support text, image input, text output, multilingual, and vision capabilities.
- 可通过 Responses API 和 Client SDKs 使用。
  Available via the Responses API and Client SDKs.
