## 核心要点

- 模型选择建议：不确定时首选旗舰模型 `gpt-5.4`（适合复杂推理和编码）。
- 若对延迟和成本敏感，可选 `gpt-5.4-mini` 或 `gpt-5.4-nano`。
- 最新模型统一支持文本、图像输入、文本输出、多语言和视觉能力。
- 可通过 Responses API 和 Client SDKs 调用。
