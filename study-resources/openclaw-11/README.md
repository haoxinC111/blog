## 核心要点

OpenClaw 是一个开放的 Agent 平台，运行在你的机器上，并支持在 WhatsApp、Telegram、Discord、Slack、Teams 等聊天应用中使用。文章讲述了品牌命名历程、产品定位（你的助手、你的机器、你的规则）、本次发布的新功能，以及未来的路线图。
