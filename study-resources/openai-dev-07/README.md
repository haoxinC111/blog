## 核心要点

- Codex App 是 OpenAI 推出的桌面端应用，用于并行处理 Codex 线程。
- 内置 worktree 支持、Git 工具和自动化能力。
- 当前支持 macOS (Apple Silicon)。
- 主要特性包括：跨项目多任务、内置终端、本地环境定义、IDE 扩展同步以及 MCP 支持。
