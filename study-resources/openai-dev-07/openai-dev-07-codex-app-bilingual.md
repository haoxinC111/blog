## 简介

Codex app 是一个专注于桌面端的应用，用于并行处理 Codex 线程，内置 worktree 支持、自动化和 Git 功能。支持 macOS (Apple Silicon)。
The Codex app is a desktop-focused application for running Codex threads in parallel, with built-in worktree support, automations, and Git capabilities. It supports macOS (Apple Silicon).
