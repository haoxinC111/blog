## 核心要点 / Key Takeaways

Claude Code 是一个 AI 驱动的编程助手，帮助你构建功能、修复错误和自动化开发任务。它能理解整个代码库，可以跨多个文件和工具工作。

Claude Code is an AI-powered coding assistant that helps you build features, fix bugs, and automate development tasks. It understands your entire codebase and can work across multiple files and tools.

### 功能 / Features

- Remote Control：从手机或其他设备继续本地会话
  Remote Control: continue local sessions from your phone or another device
- Channels：从 Telegram、Discord、iMessage 或 webhook 推入会话
  Channels: push into sessions from Telegram, Discord, iMessage, or webhook
- Cloud/Web scheduled tasks：定时运行任务
  Cloud/Web scheduled tasks: run tasks on a schedule
- GitHub Actions / GitLab CI/CD 集成
  GitHub Actions / GitLab CI/CD integration
- GitHub Code Review：自动 PR 代码审查
  GitHub Code Review: automated PR code review
- Slack 集成 — Slack integration
- Chrome 调试 — Chrome debugging
- Agent SDK：构建自定义 agent
  Agent SDK: build custom agents

### 配置通用性 / Configuration Portability

CLAUDE.md 文件、settings 和 MCP servers 在所有环境中通用。
CLAUDE.md files, settings, and MCP servers are shared across all environments.
