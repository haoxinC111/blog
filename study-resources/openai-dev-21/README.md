## 核心要点

本文介绍如何为 LLM 应用设计有效的自定义评估（Evals）。核心内容包括：
- 为什么传统软件测试对生成式 AI 不够充分
- Eval 的类型与反模式
- 完整的 Eval 工作流（定义目标、收集数据、定义指标、运行比较、持续评估）
- 摘要与文档问答的指标示例
- 利用 LLM 的判别能力（比较、分类、评分）优化评估
- 使用 o3 和 GPT-4.1 生成多样化测试数据的最佳实践
