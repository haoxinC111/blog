# Agent Evals | OpenAI API

- **Source URL**: https://developers.openai.com/api/docs/guides/agent-evals/
- **整理日期**: 2026-04-07

The OpenAI Platform offers a suite of evaluation tools to help you ensure your agents perform consistently and accurately.

OpenAI 平台提供了一套评估工具，帮助你确保智能体的表现一致且准确。

For identifying errors at the workflow-level, we recommend our [trace grading](/api/docs/guides/trace-grading) functionality.

对于识别工作流级别的错误，我们推荐使用[trace grading](/api/docs/guides/trace-grading)功能。

For an easy way to build and iterate on your evals, we recommend exploring [Datasets](/api/docs/guides/evaluation-getting-started).

为了简便地构建和迭代你的评估，我们推荐探索 [Datasets](/api/docs/guides/evaluation-getting-started)。

If you need advanced features such as evaluation against external models, want to interact with your eval runs via API, or want to run evaluations on a larger scale, consider using [Evals](/api/docs/guides/evals) instead.

如果你需要高级功能，例如针对外部模型进行评估、希望通过 API 与评估运行交互，或者希望大规模运行评估，请考虑改用 [Evals](/api/docs/guides/evals)。
