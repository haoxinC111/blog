# What is the Model Context Protocol (MCP)?

**Source URL:** https://modelcontextprotocol.io/docs/getting-started/intro  
**整理日期:** 2026-04-07

---

MCP (Model Context Protocol) is an open-source standard for connecting AI applications to external systems.

MCP（Model Context Protocol）是一种将 AI 应用连接到外部系统的开源标准。

Using MCP, AI applications like Claude or ChatGPT can connect to data sources (e.g. local files, databases), tools (e.g. search engines, calculators) and workflows (e.g. specialized prompts)—enabling them to access key information and perform tasks.

借助 MCP，像 Claude 或 ChatGPT 这样的 AI 应用可以连接到数据源（例如本地文件、数据库）、工具（例如搜索引擎、计算器）和工作流（例如专用提示词）——使它们能够访问关键信息并执行任务。

Think of MCP like a USB-C port for AI applications. Just as USB-C provides a standardized way to connect electronic devices, MCP provides a standardized way to connect AI applications to external systems.

可以把 MCP 想象成 AI 应用的 USB-C 接口。正如 USB-C 为电子设备提供了一种标准化的连接方式，MCP 也为 AI 应用连接外部系统提供了标准化的方式。

## What can MCP enable?

## MCP 能实现什么？

* Agents can access your Google Calendar and Notion, acting as a more personalized AI assistant.

  Agent 可以访问你的 Google Calendar 和 Notion，从而成为更个性化的 AI 助手。

* Claude Code can generate an entire web app using a Figma design.

  Claude Code 可以根据 Figma 设计稿生成完整的 Web 应用。

* Enterprise chatbots can connect to multiple databases across an organization, empowering users to analyze data using chat.

  企业聊天机器人可以连接组织内的多个数据库，让用户通过对话分析数据。

* AI models can create 3D designs on Blender and print them out using a 3D printer.

  AI 模型可以在 Blender 中创建 3D 设计，并用 3D 打印机将其打印出来。

## Why does MCP matter?

## 为什么 MCP 很重要？

Depending on where you sit in the ecosystem, MCP can have a range of benefits.

根据你在生态系统中的位置，MCP 可以带来多种好处。

* **Developers**: MCP reduces development time and complexity when building, or integrating with, an AI application or agent.

  **开发者**：MCP 减少了构建或集成 AI 应用/Agent 时的开发时间和复杂度。

* **AI applications or agents**: MCP provides access to an ecosystem of data sources, tools and apps which will enhance capabilities and improve the end-user experience.

  **AI 应用或 Agent**：MCP 提供了一个数据源、工具和应用的生态系统，可增强能力并改善最终用户体验。

* **End-users**: MCP results in more capable AI applications or agents which can access your data and take actions on your behalf when necessary.

  **最终用户**：MCP 带来更强大的 AI 应用或 Agent，它们可以在必要时访问你的数据并代表你执行操作。

## Broad ecosystem support

## 广泛的生态系统支持

MCP is an open protocol supported across a wide range of clients and servers. AI assistants like [Claude](https://claude.com/docs/connectors/building) and [ChatGPT](https://developers.openai.com/api/docs/mcp/), development tools like [Visual Studio Code](https://code.visualstudio.com/docs/copilot/chat/mcp-servers), [Cursor](https://cursor.com/docs/context/mcp), [MCPJam](https://docs.mcpjam.com/getting-started), and [many others](/clients) all support MCP — making it easy to build once and integrate everywhere.

MCP 是一个开放协议，得到了众多客户端和服务器的支持。像 [Claude](https://claude.com/docs/connectors/building) 和 [ChatGPT](https://developers.openai.com/api/docs/mcp/) 这样的 AI 助手，以及 [Visual Studio Code](https://code.visualstudio.com/docs/copilot/chat/mcp-servers)、[Cursor](https://cursor.com/docs/context/mcp)、[MCPJam](https://docs.mcpjam.com/getting-started) 和[许多其他工具](/clients)都支持 MCP——让你可以一次构建，随处集成。
