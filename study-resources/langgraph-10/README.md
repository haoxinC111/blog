# langgraph-10: LangGraph Overview

- **来源 URL**: https://docs.langchain.com/oss/python/langgraph/overview
- **标题**: LangGraph Overview

## 核心要点

LangGraph 是一个低级别的编排框架和运行时，用于构建长期运行、有状态的 Agent。文档涵盖安装示例、核心优势（持久化执行、人机协同、全面记忆、LangSmith 调试、生产级部署）以及 LangGraph 生态系统。
